# 🎓 Student Manager App

A simple and efficient command-line application developed in Python for managing student records. This project enables users to add student information, view stored records, and check pass/fail status based on marks.

## 📌 Overview

The Student Manager App is a beginner-friendly Python project designed to demonstrate the use of dictionaries, loops, conditional statements, and user input handling. It provides a menu-driven interface for managing student data in a simple and organized way.

## ✨ Features

* Add student names and marks
* View all student records
* Check individual student results
* Pass/Fail evaluation based on marks
* User-friendly command-line interface
* Efficient data storage using Python dictionaries

## 🛠️ Built With

* **Python**
* Dictionaries
* Loops
* Conditional Statements
* Input/Output Operations

## 🚀 Getting Started

### Prerequisites

* Python 3.x installed on your system

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/student-manager-app.git
```

Navigate to the project directory:

```bash
cd student-manager-app
```

Run the application:

```bash
python student_manager.py
```
